Welcome to Game Editor

Windows:

In order to execute Game Editor, just click on the gameEditor.exe file.

If you got crashs when open and close menus, this may indicate a compatibility problem with Game Editor and the OpenGL driver of your computer.
In this case, rename the file _noopengl in the Game Editor folder to noopengl.


Linux:

In order to execute Game Editor on Linux type (from a terminal window):
chmod +x gameEditor
./gameEditorLinux
