Jump Statements 


jump-stmt:

goto id ;

continue ;

break ;

return [expr] ;

The goto jump statement is used to redirect program flow to the target identifier, which must be a label, see: § 3.24.2, pg: . The continue and the break statements may appear only in an iteration statement. While the return statement can appear anywhere within the body of a function.
