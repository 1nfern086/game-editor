stdbool.h

The header <stdbool.h> defines four macros and a program is permitted to undefine and redefine these macros

bool
Macro which expands to type _Bool (§ 3.12.1, pg: ).

The remaining three macros are suitable for use in #if preprocessing directives. They are

true
Macro true expands to the decimal constant 1,

false

Macro false expands to the decimal constant 0.

__bool_true_false_are_defined
Macro expands to the decimal constant 1
