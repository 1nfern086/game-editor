Continue Statement

In an iteration loop the continue statement forces the start of the next iteration or loop-continuation of the inner most while, do...while or for loop. In a for loop, the next iteration resumes only after evaluation of the current iteration expr34#34: 

for(x = 10; x > 0; x = x - 1) {
     if(x > 5)  
        continue;
     printf("x = %d\n",x);
}

Only allows the numbers 5,4,3,2 and 1 to be printed out.
