Iteration Statements

C provides just three basic kinds of loops: 


iter-statement:

while ( expr ) stmt

do  stmt while ( expr ) ;

for  ( [expr32#32] ; [expr33#33] ; [expr34#34];) stmt

The conditional expr in each iteration statement must be of arithmetic or pointer type. With respect to the for loop, the conditional statement is expr33#33, which is also optional.
