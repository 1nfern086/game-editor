Boolean type: _Bool

The _Bool specifier is an object that is large enough to store the values 0 and 1. The rank of _Bool is less than the rank of all other standard integer types. Therefore, results of binary operations will always result in type that is _Bool or greater. When any scalar value is converted to type _Bool, the result is 0 if the value of the scalar equals 0; otherwise, the result is 1 -- see also the specifications for the standard header file <stdbool.h> (§ 4.1.10, pg: ).
