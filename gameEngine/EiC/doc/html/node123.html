assert.h

The header files <assert.h> defines the following macro:

assert
Synopsis: 

#include <assert.h>
void assert(int expression);

If expression is false, a message is printed to stderr and abort is called to terminate execution. However, EiC does not call abort. The source file and line number in the output message comes from the preprocessor macros __FILE__ and __LINE__. If NDEBUG is defined when <assert.h> is included, the assert macro is ignored.
