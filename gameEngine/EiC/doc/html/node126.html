float.h

In C, floating point values are represented in the normalised form:

37#37

where,
38#38 sign (29#291)
28#28 is the base or radix, typically 2, 8, 10 or 16.
39#39 the exponent, a value between a minimum 40#40 and a maximum 41#41.
42#42 precision (the number of base-28#28 digits in the significand).
43#43 the significant digits.

The header file <float.h> declares the following macros:

FLT_RADIX
radix of exponent representing 28#28.
FLT_ROUNDS
Indicates the rounding mode for floats point values:
-1, undetermined
0 toward zero
1 to the nearest
2 towards positive infinity
3 towards negative infinity

FLT_DIG
number of digits of precision in a float
FLT_EPSILON
smallest number x such that 1.0 + x != 1.0

FLT_MANT_DIG
the number of base-b digits in the floating-point significand.

FLT_MAX
maximum floating-point number
FLT_MAX_10_EXP
maximum 44#44 such that 45#45 is representable
FLT_MAX_EXP
The maximum 46#46 such that 47#47 is representable

FLT_MIN
minimum normalised floating-point number
FLT_MIN_10_EXP
minimum 46#46 such that 48#48 is a normalised number.
FLT_MIN_EXP
minimum 46#46 such that 49#49 is a normalised number.

DBL_DIG
number of digits of precision in a double
DBL_EPSILON
smallest number x such that 1.0 + x != 1.0

DBL_MANT_DIG
the number of base-b digits in the floating-point significand.
DBL_MAX
maximum double floating-point number
DBL_MAX_10_EXP
maximum 44#44 such that 45#45 is representable
DBL_MAX_EXP
The maximum 46#46 such that 47#47 is representable
DBL_MIN
minimum normalised double floating-point number
DBL_MIN_10_EXP
minimum 46#46 such that 48#48 is a normalised number.
DBL_MIN_EXP
minimum 46#46 such that 49#49 is a normalised number.
