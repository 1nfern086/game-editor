Installation

If you dowloaded the binary distribution, then you are ready to run after unzipping or un-taring the files. If you downloaded a source distribution, go to the Build page.

Before you can execute Kyra, you must have SDL properly installed and have the SDL run time libraries on the dynamic library path. Please see the SDL page, or look at the SDL FAQs.

The binary version will put files into either the KyraBinLinux or KyraBinWin directory. Simply run from Windows Explorer or the Linux command line. You can run from the command line by typing './krdemo' in KyraBinLinux or 'krdemos' in KyraBinWin directory. You should then see the Demo run.

The demo screens will automatically switch, but you can press spacebar to advance to the next test.

Note that the -f option will run the demo full screen.
