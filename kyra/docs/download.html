Download

Kyra is build on top of SDL, a cross platform game development library. It is tested on Windows and Linux. (Hopefully someone will do a Be and Mac version soon.) It has only been tested on an i86 machine.

You will need SDL 1.2 or later to run Kyra, as well as SDL_image 1.2 or later. Previous versions of SDL and SDL_image will not work with Kyra. Both of these packages are on the SDL web site and are licensed under the LGPL.

Kyra is available under the GPL license. Please read the license before downloading or using Kyra. Alternatively, you may use the less restrictive LGPL license provided you display a splash screen. Details of using the LGPL are at the top of any source file.

Feedback! Feedback is the sustenance of an open source project! Please check out the Kyra on SourceForge for the links to the mail list and discussion forums.

You can read the changes between versions, and the release notes.

Kyra 1 vs Kyra 2

Kyra 2 is an evolution of the API and features. There are some API changes, but it is generally friendly to the previous API with minor tweaks. The exception is widgets: they are completely redone in version 2. If you're starting with Kyra, start with Kyra version 2.x.

Download Kyra. In one of 2 packages:

Kyra and Kyra2 as Source, compiled on Win32 and Linux. Includes C++ code, documentation, demo code, tutorial, and tools.

Binary, a demo program for Windows or Linux. A ready to run program that demonstrates and tests Kyra features.

Note: The windows download dynamically loads the MSCRT.dll, which should be in 98 and later windows systems. (The source version works with all SDL supported systems.)
